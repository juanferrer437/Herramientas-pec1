Para la realizacion de esta práctica se han contado con multitud de herramientas, algunas de ellas se presentan a continuacion:

Visual Studio Code

Visual Studio Code es un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web.

Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código

Enlace a la web

Parcel.js

Parcel es una herramienta Javascript que procesa todo el código de nuestro sitio o aplicación web, recorriendo todos sus archivos enlazados, y generando una nueva colección de ficheros, más apropiados para el navegador.

Está en la misma categoría webpack, con una propuesta de valor diferente. Parcel promete hacer muchas cosas sin ninguna configuración y también ser rápido.

Enlace a la web

Cookie Consent

Cookie Consent es un complemento ligero de JavaScript para alertar a los usuarios sobre el uso de cookies en su sitio web.

Está diseñado para ayudarlo a cumplir rápidamente con la Ley de cookies de la UE, CCPA, GDPR y otras leyes de privacidad. Lo hicimos rápido, gratis y relativamente indoloro.

Tiny Slider

Es un paquete que implementa un carrusel de imagenes (slider) en pantalla mediante Vanilla JS, sin utilizar jQuery para ello.
